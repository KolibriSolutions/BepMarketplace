{% extends 'GenericForm.html' %} {% block formheader %} A user cannot have more than two groups. In most cases, one group is sufficient.

The following group types are available

Name Rights
type1staff 'Responsible Staff' for a project, can upgrade a proposal up to status 3. Type1staff can also be assistant to a proposal, like type2staff. Type1staff can also be trackhead. Trackheads can upgrade proposals to status 4 (public / active).
type2staff unverified 'Assistant' for a project, the daily supervisor. This is what new users automatically become on first login. The user can later be verified to normal type2staff, meaning they are approved as type2staff. It is not possible to assign this group to a user.
type2staff 'Assistant' for a project, the daily supervisor.
type3staff Support user, makes the distributions and is able to make do all kind of management tasks. Be careful, this group gives the users a lot of power!
type4staff Capacity group administration. Can see all projects for their capacity group. Edit the capacity group in Capacity Group administration
type5staff Study advisor. Can see all projects.
type6staff Professional skills administration. Can view and edit all professional skills.
