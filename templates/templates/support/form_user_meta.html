User Meta contains the additional information for a student. This includes information from Canvas and from Osiris.

Explanation

Field Explanation
Study Set by manual import from Osiris or using this form.
If this field is set to 'Automotive', the student can be given higher priority on automotive projects during automatic distribution.
Cohort Set by manual import from Osiris or using this form
Start year of the student. The later start year, the more priority in distribution.
ECTS Set by manual import from Osiris or using this form
Number of credits of the student. The higher ECTS, the more priority in distribution.
Set to zero if not yet known.
TimeSlots Time slot where this user is active. If a user logs in via the Canvas module, the current timeslot is added to this list. Time slots are never automatically removed. A student can only login to the marketplace if the user is active in the current timeslot. As long as previous Canvas BEP courses are closed for students, this prevents students who already finished their BEP to login.
Overruled Set to true to not update the 'EnrolledBEP' and 'EnrolledExt' on Canvas login.
EnrolledBEP Set during Canvas login of a user if 'Overruled' is False.
Set to True if the user logs in via Canvas course {{ settings.COURSE_CODE_BEP }} or {{ settings.COURSE_CODE_EXT }}. If this is False, the user cannot login to the system directly (without Canvas). If 'Overruled' is True and 'EnrolledBEP' is False, the user can never login.
EnrolledExt Set during Canvas login of a user if 'Overruled' is False.
Set to True if a user logs in via Canvas module of course {{ settings.COURSE_CODE_EXT }}.
